# Installation & Configuration Guide

This guide explains how to set up and deploy the ESP32-CAM Email Intrusion Detection System.

---

## Requirements
- ESP32-CAM (AI Thinker / ESP32-CAM-MB)
- Arduino IDE
- ESP32 Board Package
- ESP Mail Client Library
- Gmail account with App Password enabled

---
